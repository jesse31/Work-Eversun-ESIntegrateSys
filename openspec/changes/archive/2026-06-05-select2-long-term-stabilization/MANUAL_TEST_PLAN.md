## 異常排查指南

### 問題 1：下拉選項仍為空

**症狀**：下拉點開後無選項，或顯示「undefined」

**排查步驟**：
```javascript
// F12 Console 執行
console.log('classificationOptions:', app.$data.classificationOptions);
console.log('maintenanceResultOptions:', app.$data.maintenanceResultOptions);
```

**可能原因**：
- [ ] AJAX 請求失敗 → 檢查網路、後端 API 端點
- [ ] API 返回格式不符 → 檢查 API 是否返回 `{ Value, Text }` 格式
- [ ] Vue 資料未初始化 → 檢查 mounted() 是否執行

---

### 問題 2：選擇後值不同步

**症狀**：選擇選項後，formData 未更新

**排查步驟**：
```javascript
// F12 Console 執行
app.$data.formData.Classification = '1';  // 手動設定
// 檢查 select 是否隨之變化
```

**可能原因**：
- [ ] v-model 綁定錯誤 → 檢查 HTML 中 v-model 是否正確
- [ ] 資料類型不匹配 → 檢查 Value 是否為字符串或數字
