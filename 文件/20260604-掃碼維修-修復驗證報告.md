### 2.1 Select2 非同步初始化修復

#### Requirement: Select2 插件非同步初始化保障

**規格**: 系統應確保在 jQuery 和 Select2 CDN 完全加載後，再嘗試初始化下拉欄位的 Select2 插件

**實作位置**: `ScanRepair.cshtml:303-317`

```javascript
initializeSelect2WithFallback() {
    if (window.$ && window.$.fn.select2) {
        this.loadDropdownOptions();
    } else {
        setTimeout(() => {
            if (window.$ && window.$.fn.select2) {
                this.loadDropdownOptions();
            } else {
                console.warn('Select2 加載失敗，下拉選單將回退為原生 HTML select');
            }
        }, 100);
    }
}
```

**驗證**:
- ✅ 檢查 `window.$ && window.$.fn.select2` 的存在性 (line 304)
- ✅ 若存在則直接調用 loadDropdownOptions() (line 306)
- ✅ 若不存在則設定 setTimeout 輪詢 (line 309)
- ✅ 等待時間 100ms，最多重試 1 次（共 100ms）
- ✅ 超時後記錄警告並允許降級（line 313）

**場景覆蓋**:
1. ✅ 頁面載入時 Select2 已可用 → 直接初始化（line 304-306）
2. ✅ Select2 初始化成功 → 通過 loadDropdownOptions 載入選項（line 319-361）
3. ✅ Select2 初始化逾時 → 記錄警告、降級為原生 select（line 313）
