## 實作概要

### 修改內容
- 移除舊的 `initializeSelect2WithFallback()` 方法（單次檢查，易失敗）
- 新增 `ensureSelect2Ready(timeout = 1000)` 方法（非同步輪詢，50ms 間隔）
- 新增 `initializeSelect2Element(selector, options)` 方法（安全初始化）
- 改造 `mounted()` 生命週期（使用 setTimeout(..., 0) 非同步啟動）
- 在 Classification 和 MResult AJAX success callbacks 中新增守衛檢查

### 核心邏輯
1. **頁面載入**: mounted() → 焦點設定 → setTimeout(..., 0) 啟動非同步初始化
2. **Select2 準備**: ensureSelect2Ready() 輪詢 50ms 間隔，1000ms 超時
3. **數據加載**: 若 Select2 就緒，loadDropdownOptions() 取分類和維修原因
4. **降級防護**: 每個 AJAX callback 都有守衛檢查，若 Select2 未就緒則跳過
5. **原生 select**: 若 Select2 超時，原生 HTML select 仍可正常使用
