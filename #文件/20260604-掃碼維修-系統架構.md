# 掃碼維修功能 - 系統架構文件

## 功能概述

掃碼維修是 ESIntegrateSys 料槍維修模組的 MVP 功能，用以優化維修開單流程。操作員通過掃碼槍或手動輸入料槍編號，系統自動查詢並帶出料槍基本資訊和最新待維修紀錄，減少人為操作步驟和輸入錯誤。

## API 規格

### API Endpoint 1: GetGunByBarcode

**URL**: `/MaterialGun/GetGunByBarcode`

**Method**: `POST`

**Content-Type**: `application/x-www-form-urlencoded` 或 `application/json`

#### 請求 (Request)

```
POST /MaterialGun/GetGunByBarcode HTTP/1.1
Content-Type: application/x-www-form-urlencoded

barcode=ABC001
```

或使用 jQuery AJAX:

```javascript
$.ajax({
    url: '/MaterialGun/GetGunByBarcode',
    type: 'POST',
    dataType: 'json',
    data: { barcode: 'ABC001' },
    success: function(response) { ... }
});
```

#### 參數 (Parameters)

| 參數名 | 型別 | 必填 | 說明 |
|--------|------|------|------|
| barcode | string | ✅ | 掃入的料槍編號（條碼值） |

#### 回應 (Response)

##### 成功情況

```json
{
  "success": true,
  "message": "查詢成功",
  "data": {
    "repairSno": 12345,
    "Eno": "E001",
    "Sno": "ABC001",
    "Trade": "Brand X",
    "Size": "10mm"
  }
}
```

| 欄位 | 型別 | 說明 |
|------|------|------|
| success | boolean | 查詢是否成功 |
| message | string | 結果訊息 |
| data | object | 料槍與維修資訊 |
| data.repairSno | integer | 維修紀錄序號（用於後續提交） |
| data.Eno | string | 設備編號 |
| data.Sno | string | 料槍編號 |
| data.Trade | string | 廠商 |
| data.Size | string | 型號 |

##### 失敗情況 - 料槍不存在

```json
{
  "success": false,
  "message": "查無料槍編號：ABC001"
}
```

##### 失敗情況 - 料槍無待維修紀錄

```json
{
  "success": false,
  "message": "料槍 ABC001 無待維修紀錄或已全部完修"
}
```

##### 失敗情況 - 輸入為空

```json
{
  "success": false,
  "message": "請輸入料槍編號"
}
```

##### 失敗情況 - 伺服器錯誤

```json
{
  "success": false,
  "message": "伺服器錯誤：<異常訊息>"
}
```

### API Endpoint 2: MaterialGunRepair (POST - 複用既有)

**URL**: `/MaterialGun/MaterialGunRepair`

**Method**: `POST`

#### 請求 (Request)

```
POST /MaterialGun/MaterialGunRepair?sno=12345 HTTP/1.1
Content-Type: application/x-www-form-urlencoded

sno=12345
MaterialGun_Sno=ABC001
Classification=100
MaintenanceResult=50
Other=
ChangeItemName=Spring
ChangeItemNo=SP-001
b_Chk=
__RequestVerificationToken=<token>
```

#### 參數 (Parameters)

| 參數名 | 型別 | 必填 | 說明 |
|--------|------|------|------|
| sno | integer | ✅ | 維修紀錄序號 (from GetGunByBarcode) |
| MaterialGun_Sno | string | ✅ | 料槍編號 |
| Classification | integer | ✅ | 分類代碼 |
| MaintenanceResult | integer | ✅ | 維修原因代碼 |
| Other | string | ❌ | 其他原因說明 (when MaintenanceResult=99) |
| ChangeItemName | string | ❌ | 更換部品名稱 |
| ChangeItemNo | string | ❌ | 更換部品料號 |
| b_Chk | string | ❌ | 按鈕名稱 (維修完成/等待材料) |

#### 回應 (Response)

成功: 重導至 `/MaterialGun/MaterialGunRepairView`

失敗: 返回錯誤頁面或 JSON 錯誤回應

## 資料庫操作

### GetGunByBarcode 的資料庫查詢

#### Query 1: 驗證料槍存在

```csharp
var gunInfo = db.ES_MaterialGunInfo
    .FirstOrDefault(x => 
        x.MaterialGun_Sno == normalizedBarcode && 
        !x.MaterialGunDiscard.Value  // 排除已報廢料槍
    );

// 對應 SQL:
// SELECT TOP 1 * FROM ES_MaterialGunInfo
// WHERE MaterialGun_Sno = @normalizedBarcode 
//   AND (MaterialGunDiscard IS NULL OR MaterialGunDiscard = 0)
```

#### Query 2: 查詢最新未完修紀錄

```csharp
var repairRecord = db.ES_MaterialGunRepair
    .Where(x => 
        x.MaterialGun_Sno == normalizedBarcode && 
        x.Chk == true  // 未完修 (Chk=True)
    )
    .OrderByDescending(x => x.RepairDate)  // 按維修日期降序
    .FirstOrDefault();

// 對應 SQL:
// SELECT TOP 1 * FROM ES_MaterialGunRepair
// WHERE MaterialGun_Sno = @normalizedBarcode 
//   AND Chk = 1
// ORDER BY RepairDate DESC
```

#### 邏輯驗證

```
如果 gunInfo == null
  → 返回 "查無料槍編號"

如果 repairRecord == null
  → 返回 "無待維修紀錄"

否則
  → 返回 repairRecord.sno + gunInfo 資訊
```
